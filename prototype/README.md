# Claude Skills Marketplace - 原型设计文档

## 项目概述

基于产品需求文档（PRD），我为 Claude Skills Marketplace 创建了一个完整的原型设计。该平台是一个用于分享、发现和下载 Claude Code Skills 的市场平台。

## 设计风格

### 视觉设计
- **配色方案**：蓝紫色渐变主题（#667eea 到 #764ba2）
- **主色调**：Indigo (#3B82F6) 和 Purple (#8B5CF6)
- **字体**：
  - 标题：Space Grotesk（现代科技感）
  - 正文：DM Sans（高可读性）
- **设计风格**：现代、简洁、卡片式布局
- **响应式设计**：支持桌面和移动端

### UI 组件
- 卡片式设计with hover效果
- 渐变色图标背景
- 圆角按钮和卡片
- 阴影和过渡动画

## 技术实现

### 前端技术栈
- **HTML5**：语义化标签
- **Tailwind CSS**：通过 CDN 引入，快速原型开发
- **Font Awesome**：图标库
- **Google Fonts**：Space Grotesk 和 DM Sans

### 响应式设计
- 使用 Tailwind 的响应式类
- 移动端优先设计
- 断点：sm (640px), md (768px), lg (1024px)

### 交互设计
- Hover 状态：卡片上浮效果
- 过渡动画：平滑的颜色和变换过渡
- 模态框：安装说明弹窗
- 复制功能：一键复制安装命令

## 设计原则

### 1. 用户体验
- **简洁直观**：清晰的导航和信息架构
- **快速发现**：首页即可看到热门和分类
- **详细信息**：Skill 详情页提供完整信息
- **便捷操作**：一键下载和安装

### 2. 视觉设计
- **现代科技感**：蓝紫色渐变，符合开发者工具定位
- **卡片式布局**：信息组织清晰
- **渐变图标**：视觉吸引力强
- **一致性**：统一的设计语言

### 3. 性能优化
- **CDN 资源**：快速加载
- **响应式图片**：适配不同设备
- **最小化依赖**：仅使用必要的库

## 使用说明

1. 打开 `index.html` 查看首页
2. 点击任意 Skill 卡片可跳转到详情页
3. 所有页面都是响应式的，可以在不同设备上查看
4. 使用浏览器的开发者工具可以测试不同屏幕尺寸
